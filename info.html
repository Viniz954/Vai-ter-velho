Sobre o Site

Nossa professora de STEAM nos propôs um projeto sobre etarismo digital. Como solução, criamos este site voltado para pessoas idosas, com o objetivo de ajudá-las a evitar golpes e lidar melhor com a tecnologia no dia a dia. Este é um projeto sem fins lucrativos.

Como se tornar um apoiador?

Para se tornar um apoiador do nosso projeto é bem simple. Basta entrar no nosso repositório do Github para relatar Bugs, enviar sugestões, e editar o site.

Caso queira entrar em contato com a equipe principal, contate: equipevaitervelho@gmail.com
